Create a timeline component that merges movimentacoes and prazos into a single chronological view.

**Props interface:**
```typescript
interface ProcessoTimelineProps {
  movimentacoes: MovimentacaoRow[];
  prazos: PrazoRow[];
}
```

**Implementation:**

1. Define a unified `TimelineItem` type:
```typescript
type TimelineItem = {
  id: string;
  date: string;        // ISO date string for sorting
  type: 'movimentacao' | 'prazo';
  description: string;
  metadata: string;    // e.g., "Publicacao - PJe" or "Fatal - Prazo peremptorio"
  urgente: boolean;
  status?: string;     // prazo status (pendente/cumprido/perdido)
  teor?: string | null;
};
```

2. Merge both arrays into TimelineItem[], sort by date descending (newest first).

3. Render as a vertical timeline with a left-side line:
   - Left column: date formatted as "dd/MM" with year shown only when it changes
   - Center: vertical line (2px, `var(--color-border)`) with a circle node at each event
   - Right: event card with description, metadata, optional teor excerpt

4. Visual distinction by type:
   - Movimentacao: circle node uses `var(--color-primary)`, icon `FileText` (size 12)
   - Prazo: circle node uses `var(--color-tier-warning)` for pendente, `var(--color-success)` for cumprido, `var(--color-danger)` for perdido. Icon `Clock` (size 12)
   - Urgent items: left border-l-2 with `border-l-causa-danger`

5. Circle node: 24x24px circle with white background and colored border (2px), icon centered inside. Connected by a 2px vertical line segment to next node.

6. Date grouping: Group by month. Show month header ("Marco 2026", "Fevereiro 2026") as a separator between groups.

7. Teor: If movimentacao has `teor`, show first 2 lines with `line-clamp-2` in `text-xs-causa text-[var(--color-text-muted)]`.

8. Empty state: When both arrays are empty, show centered text "Nenhum evento registrado" with a Clock icon.

9. Use existing CAUSA design tokens: `var(--radius-md)`, `var(--shadow-sm)`, `text-sm-causa`, `text-xs-causa`, `transition-causa`.

Import icons from lucide-react: `FileText`, `Clock`, `AlertTriangle`.
Import types from `../../lib/api`: `MovimentacaoRow`, `PrazoRow`.
