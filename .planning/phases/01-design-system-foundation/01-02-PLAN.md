Replace the entire implementation with Radix Dialog base. MUST maintain the same ModalProps interface (open, onClose, title, children, size?, footer?) for backward compatibility.

Key implementation details:
- Import `* as Dialog from '@radix-ui/react-dialog'` and `{ AnimatePresence, motion } from 'motion/react'`

  - `exit={{ opacity: 0, scale: 0.95 }}`

- Close button uses `Dialog.Close asChild` wrapping a button with X icon
- `Dialog.Title` for accessible modal title
