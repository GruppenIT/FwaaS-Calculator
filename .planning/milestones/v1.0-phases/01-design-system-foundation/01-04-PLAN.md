**Badge (badge.tsx) — DS-11:**
Create a status pill component with ONLY 4 predefined statuses (no custom color prop — per CONTEXT.md decision to prevent inconsistency):

```typescript
export type BadgeStatus = 'active' | 'suspended' | 'archived' | 'closed';
```

Status-to-style mapping:
- `active` -> label "Ativo", blue: `bg-[var(--color-tier-info)]/10 text-[var(--color-tier-info)]`
- `suspended` -> label "Suspenso", amber: `bg-[var(--color-tier-warning)]/10 text-[var(--color-tier-warning)]`
- `archived` -> label "Arquivado", gray: `bg-[var(--color-text-muted)]/10 text-[var(--color-text-muted)]`
- `closed` -> label "Encerrado", verde-agua: `bg-[var(--color-success)]/10 text-[var(--color-success)]`

Component: `forwardRef<HTMLSpanElement>` with `inline-flex items-center gap-1 px-2 py-0.5 rounded-[var(--radius-sm)] text-xs-causa font-medium`. Accept optional `className` for spacing overrides.

Export both `Badge` and `BadgeStatus` type.

**Select (select.tsx) — DS-15:**
Create an accessible select dropdown using Radix Select primitives:

Props:
```typescript
interface SelectOption { value: string; label: string; }
interface SelectProps {
  value?: string;
  onValueChange?: (value: string) => void;
  options: SelectOption[];
  placeholder?: string;
  label?: string;
  error?: string;
  disabled?: boolean;
  className?: string;
}
```

Implementation:
- Wrap in `div` with optional label (same pattern as Input: `text-sm-causa font-medium text-[var(--color-text-muted)]`)
- `RadixSelect.Root` with value/onValueChange/disabled
- `RadixSelect.Trigger`: styled to match Input — `h-9 px-3 rounded-[var(--radius-md)] bg-[var(--color-surface)] border border-[var(--color-border)] text-base-causa focus-causa transition-causa` with chevron icon (ChevronDown from lucide-react)
- Error state: same as Input — `border-[var(--color-danger)]` when error prop is truthy
- `RadixSelect.Portal` > `RadixSelect.Content` with `position="popper"` and `sideOffset={4}`:
  - Content styles: `bg-[var(--color-surface)] border border-[var(--color-border)] rounded-[var(--radius-md)] shadow-[var(--shadow-md)] p-1 z-50`
- `RadixSelect.Item` for each option: `px-3 py-2 text-sm-causa rounded-[var(--radius-sm)] cursor-pointer outline-none data-[highlighted]:bg-[var(--color-surface-alt)]`
- Error message below trigger (same as Input pattern)
