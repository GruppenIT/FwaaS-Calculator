### Cross-Phase Wiring Issues

#### HIGH — Missing causa-info token (visual defect)

**From:** `globals.css` @theme inline block
**To:** `clientes-page.tsx` line 17, `cliente-detail-page.tsx` line 58
**Impact:** Prospecto client status badge renders with no background and no text color — invisible text on transparent background.
**Fix:** Add `--color-causa-info: var(--color-tier-info);` to `@theme inline` block and corresponding `:root`/`.dark` entries.
**Affected:** DS-01, LIST-05, DS-11

#### MEDIUM — Timezone-unsafe diasRestantes duplicate

**From:** `processo-detail-page.tsx` lines 115-120 (local copy)
**To:** `prazo-countdown.tsx` (fixed UTC-safe version)
**Impact:** In UTC-3 (Brazil), prazos due "today" may show as overdue by 1 day in the Prazos tab of processo detail. The listing page uses the correct version.
**Fix:** Remove local `diasRestantes()`, import from `prazo-countdown.tsx`.
**Affected:** DET-01, LIST-02

#### LOW — 5 DS components orphaned

**Components:** Select, Checkbox, Textarea, Card, StatusDot
**Impact:** Built and exported in Phase 1 but never consumed. Modals re-implement native equivalents inline. Not a runtime defect but represents design system adoption debt.
**Affected:** DS-12, DS-13, DS-15, DS-16, DS-17
